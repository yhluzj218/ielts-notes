# G008 — Modal Verbs + Bare Infinitive (Not Past Form)

## Core Rule

Modal verbs (`might`, `could`, `would`, `should`, `can`, `will`, `may`, `must`) are always followed by the **bare infinitive** (base form of verb — no `-ed`, no `-s`, no `-ing`).

> Modal + base verb: `might go`, `could help`, `would improve`

A related error: `end up` always takes a **gerund** (`-ing`), not an infinitive.

## Why it matters in IELTS

Modal verbs are essential in Task 2 writing for making suggestions and predictions. Wrong form after a modal is visible and directly affects GRA.

## Common mistakes

❌ `This might ended up costing more.`
❌ `The government could implemented new policies.`
❌ `We should improved the system.`
❌ `People end up go for work in a different city.` (`end up` + infinitive — wrong)

## Good examples

✅ `This might end up costing more.`
✅ `The government could implement new policies.`
✅ `We should improve the system.`
✅ `People end up working in a different city.` (`end up` + gerund)

**`end up` patterns:**
✅ `end up + -ing`: `She ended up studying abroad.`
✅ `end up + noun`: `He ended up a doctor.`
❌ `end up + to do`: wrong

## Reusable patterns

```
Modal + bare infinitive:
could [help / reduce / improve / address / increase]
might [lead to / result in / cause]
should [be / consider / focus on]
would [benefit / allow / enable]

end up + gerund:
end up [living / working / paying / losing]
```

## Mini practice

Fix the errors:

1. `The policy might reduced unemployment rates.`
2. `Students could easily finds information online.`
3. `Many people end up to work in fields unrelated to their degree.`

**Answers:**
1. The policy might **reduce** unemployment rates.
2. Students could easily **find** information online.
3. Many people end up **working** in fields unrelated to their degree.
