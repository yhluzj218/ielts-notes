## Task 1 重點（訓練課程回饋信 v2）

| 主題 | 老師的 Feedback |
|------|----------------|
| feedback 不可數 | `a short feedback` → `some feedback` 或 `a piece of feedback` |
| 時態 | `I have participated in` → `I participated in`（last weekend = 過去式）|
| 形容詞 vs 副詞 | `made it easily` → `made it easy`（easy 是形容詞補語）|
| 過去被動 | `we been separated` → `we were separated into different groups` |
| handle skills | 老師不理解；建議改 `conflict resolution skills` 或 `customer service skills` |
| although + however | 只能留一個連接詞 |
| 結尾措辭 | `I am glad to share more if you like` → `I would be happy to share any further details` |
| Bullet 3 要解釋 | 說「對經理不相關」但沒有解釋**為什麼**；要補一句理由 |
| 內容要更真實 | 加入具體跟 McDonald's 工作相關的細節；例如：`As we frequently deal with difficult customers in the evening, the training session on managing challenging situations was really useful.` |

### 信件語氣（正式度）說明

## Task 2 重點（學費議論文 v2）

| 主題 | 老師的 Feedback |
|------|----------------|
| due to 用法 | `due to` ≠ `because`；要接子句用 `because` 或 `due to the fact that` |
| reasons 數量 | 說了 "two main reasons" 但寫了三個 body paragraph；說幾個就寫幾個 |
| end up 用法 | `end up going for work` → `end up working rather than continuing their studies`（work 是動詞不是名詞） |
| a + 複數 | `will have a long-term benefits` → `will have long-term benefits`（a + 複數矛盾）|
| Para 2 展開不足 | 點出「是負擔」，馬上跳到創業者例子——中間缺少解釋；應補：「學費通常需要三個月薪資，而學生沒有收入」→ 這個解釋本身就能撐一整段 |
| Para 3 例子後要收尾 | Cambridge 例子後要補一句說明：`education is the only way to reverse the class divide`，才算完整 |
| Para 4 太複雜 | 開頭要先用簡單句幫讀者理解方向，再解釋細節：`Education is so important that it should not be treated like other products.` |
| a part of a portion | 同義重複，只用一個 |
| 正面回應題目前提 | 題目說「degree 對個人好處大於社會」→ 要直接回應這個前提；老師建議：說「society benefits as much as individuals」並舉具體例子（doctors, teachers）|

| 情境 | 建議 |

## 本堂確認的錯誤

| 原句 | 問題 | 修改 |
|------|------|------|
| `a long-term benefits` | a + 複數矛盾 | `long-term benefits` |
| `end up going for work` | work 是動詞不是名詞 | `end up working rather than continuing their studies` |
| `a part of a portion of` | 同義重複 | `part of` 或 `a portion of`，選一個 |
| `due to a college certificate benefit them` | due to 後不能接子句 | `because a college certificate benefits them` |
