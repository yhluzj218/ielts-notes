# E003 — Linking Devices Overview

## Core Rule

Linking devices signal the **relationship** between ideas. Using the wrong type (e.g., concession where you need addition) confuses the reader.

## By Function

### Adding / Reinforcing
```
Furthermore, ...
Moreover, ...
In addition, ...
Additionally, ...
Not only X, but also Y.
```

### Contrasting / Conceding
```
However, ...                    (new sentence)
Nevertheless, ...               (new sentence)
Although [clause], [clause].   (same sentence)
While [clause], [clause].      (same sentence)
On the other hand, ...
In contrast, ...
Despite [noun phrase], ...
```
→ See [E001](E001_concession_expressions.md) for full concession toolkit

### Cause / Result
```
Therefore, ...
As a result, ...
Consequently, ...
This has led to ...
Due to [noun phrase], ...
Because [clause], ...
```
→ See [E002](E002_cause_result_expressions.md) for full cause/result toolkit

### Giving Examples
```
For example, ...
For instance, ...
To illustrate, ...
A clear example of this is ...
This can be seen in ...
```

### Sequencing / Structuring
```
First, ... / Firstly, ...
Second, ... / Secondly, ...
Finally, ...
To begin with, ...
To conclude, ...
```

### Opinion / Position (Task 2)
```
This essay will argue that ...
I strongly believe that ...
It is my view that ...
From my perspective, ...
```

## What to avoid

❌ Overusing `In addition` for every paragraph (repetitive — loses CC marks)
❌ Starting every sentence with a connector (reads like a list, not an essay)
❌ `Nowadays, more and more people...` (clichéd opening)
❌ `In another hand` → ✅ `On the other hand` (see [P002](../prepositions/P002_on_the_other_hand.md))
