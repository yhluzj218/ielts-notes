# E004 — Part 3 Opinion Openers

## Core Rule

Part 3 requires **society-level opinions** — not personal experiences. The openers below signal this register.

Use these instead of `I personally...` or `In my personal experience...` (which belong to Part 1/2).

## Toolkit

### Opening your answer

```
"I think most people would agree that..."
"In general, society tends to..."
"From a societal perspective, ..."
"It seems to me that the main issue here is..."
"I believe the key factor is..."
"There's a general feeling that..."
"Most experts / researchers would argue that..."
```

### Giving reasons

```
"This is largely because..."
"The main reason is that..."
"A key factor is..."
"This stems from..."
"This can be attributed to..."
```

### Adding examples / evidence

```
"For example, in many countries..."
"A clear illustration of this is..."
"We can see this in..."
"Research suggests that..."
"Statistics show that..."
```

### Extending / implications

```
"In the long run, ..."
"Over time, this is likely to..."
"The long-term impact of this would be..."
"If this continues, ..."
"This could potentially lead to..."
```

### Conceding / balancing

```
"Of course, there are cases where..."
"That said, ..."
"While this is generally true, ..."
"It depends to some extent on..."
```

## Reusable sentence patterns (drop-in templates)

```
I think [claim about society]. This is because [societal reason]. For example, [general trend/example]. In the long run, [implication].
```

**Example:**
> "I think governments carry the greater responsibility. This is because individual actions alone rarely achieve population-scale change. For example, unless regulations push companies to reduce emissions, most people simply continue their habits. In the long run, structural change driven by policy tends to be far more effective than relying on individual willpower."
