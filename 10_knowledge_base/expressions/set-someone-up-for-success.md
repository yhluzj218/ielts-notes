## Expression

`set someone up for success / failure`

### Function

Describe how a prior condition (an environment, a rule, a piece of preparation)
makes a later outcome easier or harder for someone to achieve — shifts the
explanation from "this person lacks willpower" to "the situation was designed
well/badly."

### Usage

Use when arguing that outcomes are shaped by environment/design/preparation
rather than individual effort alone — a useful reframing move in Problem/Solution
or Discuss Both Views essays about behaviour, parenting, education, or workplace
performance.

### Common Collocations

- set someone up for success
- set someone up for failure
- not setting [someone] up for success

### Example

Expecting a child to ignore a phone that is sitting in plain sight does not set
them up for success; removing the device from view makes the same expectation
realistic.

- Family & Parenting
- Education
- Work & Career
- Technology (self-control / habit design)
