## Connie's Diagnosed Reading Principles (From Own Test Data)

### Evidence Verification Before Selecting

| Field | Value |
|---|---|
| Source | `09_coach/performance/reading_db.md` — C20 T1 Q28, Q35, Q40 |
| Pattern | Keyword found → answer selected WITHOUT verifying all conditions |
| Correct process | Keyword → paragraph → ALL conditions verified → compare alternatives → select |
| Confidence | High (observed in 3 separate question types in one test) |
| Status | No teacher has yet confirmed this as their rule — but the data is consistent |

---

### Main Idea ≠ Memorable Story Detail

| Field | Value |
|---|---|
| Source | `09_coach/performance/reading_db.md` — Q40 (0%) |
| Pattern | Chose title based on an interesting passage event, not the author's overall purpose |
| Training rule | After reading: ask "What is the author trying to argue/show overall?" — not "what happened?" |
| Confidence | Medium (single data point) |
