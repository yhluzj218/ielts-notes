# Stage 3 IELTS Chunk Memory Chants

Generated chant packages are stored here:

`YYYY-MM-DD_chant_<pattern-slug>.md`

Each package contains:

1. one short repetitive IELTS chunk-memory song;
2. a five-to-eight-item substitution bank;
3. IELTS topic links;
4. one Suno-ready prompt in the selected style.

The Target Sentence Pattern must always be supplied by the user.
