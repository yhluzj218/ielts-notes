# Sentence Pattern

`The more [X] ..., the greater [Y] ...`

### Function

遞進因果：表達「投入越多，回報／效果越大」。把「原因—結果」壓進一個對稱結構，
比 `If you do more X, you get more Y` 高一級且更自然。

### Structure

`The more [noun/effort] + [subject + verb], the greater the [noun] + [subject + verb].`

- 兩半都是「the + 比較級」開頭，結構要對稱
- ⚠️ 後半句不能寫成一般句：`The more efforts you spend..., it reward you...` ❌
  （2026-07-13 原稿的錯法——結構開了頭沒接完）
- `effort` 此處不可數：`the more effort`，不是 `the more efforts`

### Usage

Body 段中段，把 mechanism 收攏成 impact 時使用。變體：`the more..., the more...` /
`the more..., the better...` / `the less..., the fewer...`。

### Example

The more effort students put into these areas, the greater the competitive
advantage they gain in the long term.

### IELTS Topics

- Education（投入與回報）· Work（練習與技能）· Health（運動與健康）· Environment（減排與效益）
