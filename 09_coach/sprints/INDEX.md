# Sprint Index

> Two sprint types exist, and they are not equal — **Skill Sprint is primary, Theme
> Sprint is supplementary.**
> - **Skill Sprint** (`TEMPLATE.md`) — 2-week, evidence-driven, skill remediation.
> Decides WHAT to practise each day. Tracked below.
> - **Theme Sprint** (`THEME_SPRINT_TEMPLATE.md`) — weekly, content-theme task
> control. Only decides WHICH TOPIC to use when a Skill Sprint day needs one; never
> adds or overrides a Skill Sprint task. Links to `10_knowledge_base/ideas/<theme>/`. Tracked
> in its own table below (no evidence/confidence columns — different structure).
