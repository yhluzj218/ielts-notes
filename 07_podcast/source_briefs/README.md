# NotebookLM Source Briefs

Generated files from the active Phase 1 workflow are stored here:

`YYYY-MM-DD_notebooklm_source_brief_<topic-slug>.md`

Each file contains a NotebookLM Source Briefing Article and a NotebookLM Generation
Prompt. Research-Assisted files also begin with a Source Log.

No file is created when the workflow returns `Source is insufficient.`
