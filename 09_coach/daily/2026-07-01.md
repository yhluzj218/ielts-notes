## 📖 晚上練習（90 min）

### Cambridge Listening S4（90 min）

**Step 1 — 看題目（10 min）**
- 讀完所有 S4 題目
- 畫出每題的「區分關鍵字」（讓各選項不同的那個字，不是共用的）
- Note Completion 題：預測每個空格填什麼詞性（名詞 / 數字 / 形容詞？）

**Step 2 — 聽音頻並作答（35 min）**
- 聽到第一個關鍵字不要馬上寫
- 注意：reversal signals（but / however / actually / not anymore / used to）
- 聽到 "oh sorry" 或自我更正 → 永遠用更正後的答案

**Step 3 — Transcript analysis（45 min）**

每一道錯誤：
1. 在 transcript 找到正確答案
2. 找到「排除錯誤選項」的字在哪裡
3. 用下面分類標注這題的錯誤原因：

| 錯誤類型 | 說明 |
|---|---|
| `word/phrase` | 沒認出 paraphrase |
| `qualifier trap` | 聽到第一個字就選了，沒等完整條件 |
| `self-correction missed` | 沒聽到 "oh sorry" 的更正 |
| `stuck on previous` | 還在寫上一題，錯過這題 |
| `concentration drop` | 長段落後注意力分散 |
| `misunderstood` | 聽到了但理解錯 |

**做完後告訴我：**
- S4 得幾分（X/10）
- 每題錯誤的分類
- 有沒有出現 reversal signal 你沒抓到的

我會更新 `listening_db.md`。
