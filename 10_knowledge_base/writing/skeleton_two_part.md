# Skeleton — Two-part Question

> Last updated: 2026-07-01

---

## 辨識規則

**只看最後一句指令：**

| 指令關鍵字 | 判斷 |
|---|---|
| 兩個問號，各自有不同焦點 | ✅ Two-part Question |
| `Why is this?` + `Is this a positive or negative development?` | ✅ Two-part Question |
| `What are the reasons?` + `Do you think this is a good thing?` | ✅ Two-part Question |

**與 Problem+Solution 的差別：** Two-part 的兩個問題不一定是「原因 + 解法」，可以是任意兩個不同角度的問題（例：原因 + 評價、影響 + 你的看法）。

**關鍵規則：** 兩個問題都要回答，深度相等。其中一個寫太短 → Task Response 直接扣分。

---

## 標準結構

```
Intro:   Paraphrase + 預告你會回答兩個問題
Body 1:  完整回答 Question 1（含 development chain）
Body 2:  完整回答 Question 2（含 development chain）
Concl:   Restate 整體觀點
```

**Intro 模板：**
> `[Paraphrase].` `This essay will address both questions in turn.`

---

## Skeleton 自我檢查

- [ ] Intro 有預告兩個問題都會回答？
- [ ] Body 1 完整回答了 Question 1（有 development chain，不是只有一句話）？
- [ ] Body 2 完整回答了 Question 2（深度和 Body 1 相當）？
- [ ] 兩個 body 沒有回答同一個問題？
- [ ] Conclusion 整合了兩個回答的重點？
