## Expression

`set the bar higher / lower`

### Function

Talk about raising or lowering standards, expectations, or requirements for a
group of people or for a system — useful when arguing that current standards
are too lenient (or too strict) and should change.

### Usage

Use when the argument is "the current expectation is too easy/hard to meet, and
it should be made stricter/more lenient." Works for policy, parenting,
education, and workplace standards.

### Common Collocations

- set the bar higher / set the bar lower
- the bar needs to be higher
- a low bar / a high bar for [something]

### Example

If schools set the bar higher for what counts as acceptable screen use during
class time, students would develop better concentration habits.
