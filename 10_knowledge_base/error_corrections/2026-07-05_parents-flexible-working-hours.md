# Error Correction — Parents and flexible working hours

## Front

parenting now having more flexible working hour slot than the case decades ago.

## Back

### Corrected Version

Working parents now have much greater flexibility in their working hours than was
the case a few decades ago.

### Error Analysis

1. **`parenting` 指的是活動，不是一群人。** `Parenting` 是「養育孩子」；當主詞是父母
   本人時，應使用 `parents` 或 `working parents`。
2. **`having` 在這裡無法構成完整主要動詞。** 句子需要限定動詞 `have`：
   `Parents now have ...`。
3. **`working hour slot` 不是表達一般工作彈性的自然搭配。** 可以使用
   `flexible working hours`、`flexible work schedules` 或
   `flexible working arrangements`。
4. **`than the case` 不完整。** 固定比較句型是 `than was the case`，不能省略
   `was`。
5. **`decades ago` 文法上可行，但較模糊。** IELTS 比較句使用 `a decade ago`、
   `two decades ago` 或 `a few decades ago` 通常更自然。
6. **句首第一個字需要大寫。** 應從 `Parents` 或 `Working parents` 開始。

### Key Reminder

主詞要用人，接完整的限定動詞，並保留完整比較結構：
`Parents now have ... than was the case a few decades ago.`
