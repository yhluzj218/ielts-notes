# Error Correction — Programming and AI prototypes

## Front

Programming used to be the preserved of qualified programmer. Now even designers
can build their prototypes by AI Agents.

## Back

### Corrected Version

Programming used to be the preserve of skilled programmers. Now even designers can
build functional prototypes using AI agents.

### Error Analysis

1. **`the preserved of` 使用了錯誤的詞形。** 固定表達是名詞片語
   `the preserve of`，意思是某個活動或領域主要由特定群體專屬。這裡不是動詞
   `preserve`，也不是過去分詞 `preserved`。
2. **`qualified programmer` 雖可理解，但不符合這裡最自然的說法。** 可以使用
   `skilled programmers`、`experienced programmers` 或
   `professional programmers`。因為你指的是一般群體，所以使用複數
   `programmers`。
3. **`by AI agents` 會讓 AI agents 變成動作執行者。** 這句中真正採取行動的是
   designers，AI agents 是工具，因此應使用 `using AI agents`。
4. **`AI Agents` 過度大寫。** `AI` 是縮寫，保留大寫；普通名詞 `agents` 應小寫：
   `AI agents`。
5. **`build their prototypes` 文法正確。** 不過 `build functional prototypes` 更具體
   自然，因為它強調做出來的原型確實可以運作。

### Key Reminder

表達某個群體專屬的領域時使用 `the preserve of`；某人利用工具完成動作時使用
`using`。
