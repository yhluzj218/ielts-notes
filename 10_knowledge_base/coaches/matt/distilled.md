## Teaching Philosophy

Matt's primary frame is task response and method accuracy before language. He consistently pushes Connie to identify the correct question type first, develop ideas fully, and write complete essays on time — not to upgrade vocabulary or polish sentences.

---

## Core Coaching Principles

| Principle | Evidence | Lesson |
|---|---|---|
| Task Response is the first priority | "切題" corrected before grammar or vocabulary | All writing lessons |
| Idea development: Point → Explain → Example → Result | Body paragraphs must complete the chain; stopping at the point scores low | 2026-06-19 |
| Agree/Disagree ≠ Positive/Negative ≠ Discuss Both | Question type must be identified correctly before planning | 2026-06-19 |
| Finished > Perfect | Complete essay beats an unfinished polished attempt | 2026-06-23 |
| Plan 5 minutes before writing | Structure first, writing second; no editing during drafting | All writing lessons |
| Grammar from own errors, not systematic study | Matt corrects specific errors from Connie's writing — no grammar workbooks | All sessions |
| Vocabulary is not the problem | Matt's assessment: "instinctively, not a vocabulary problem" — explicitly confirmed | 2026-06-08 (speaking), all writing sessions |
| Word-form control is the real vocabulary work | "You have more than enough words. You just need to learn the control of: this is a noun, and this is a verb." Learn POS + other forms with every new word — "learn one word, actually learn three words" | 2026-06-26 |
| No IELTS social media content | Explicitly warned against | 2026-06-08 |
| Shadowing for pronunciation — but after intonation lesson | Don't shadow before intonation is introduced | 2026-05-25 |

## Writing-Specific Principles

| Area | Matt's rule |
|---|---|
| Task 1 — letter | Each bullet point = its own paragraph, no exceptions |
| Task 1 — register | Informal must sound natural; Formal must be consistently respectful |
| Task 2 — structure | Introduction (paraphrase + position) → Body 1 → Body 2 → Conclusion |
| Task 2 — body paragraphs | Both body paragraphs must be equally developed |
| Task 2 — discussion essays | Even the weaker side must be argued properly — not dismissed |
| Connector usage | Study by function, not by memorisation; common errors: while/whereas, although/however |

---

## Speaking-Specific Principles

| Area | Matt's rule |
|---|---|
| Part 2 | Must fill full 2 minutes — stopping early costs marks |
| Part 3 | Answers must address society-level scope, not just personal opinion |
| Pronunciation | Too precise / too individual — target connected speech, contractions, natural rhythm |
| No memorisation | Rehearsed-sounding answers hurt the score |

---

## Anti-Patterns Matt Has Flagged

- Misreading the question type before planning
- Perfecting one paragraph while leaving others underdeveloped
- Overthinking / editing during the drafting phase
- Assuming the problem is vocabulary (confirmed: it is not)
- Using IELTS social media content
