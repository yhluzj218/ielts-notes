# SP002 — Part 3 Strategy: Society Level, Not Personal

## Core Rule

Part 3 asks about **society-level opinions** — not your personal experiences.

> Part 1 & 2: "What do YOU do / think / like?"
> Part 3: "What do PEOPLE generally think? What does SOCIETY tend to do?"

Answering Part 3 with personal stories brings the score down.

## Why it matters in IELTS

Part 3 tests abstract thinking and extended opinion. Using `people generally`, `many people`, `society tends to` signals that you understand the question type. Personal anecdotes at Part 3 level suggest a limited range.

## Answer structure (4–5 sentences)

```
Position:     "I think [claim about society/people/government]."
Reason:       "This is because [societal reason — not personal]."
Extension:    "For example, [trend / data / general behaviour]."
Implication:  "In the long run / As a result, [wider consequence]."
(Optional) Concession: "Of course, there are cases where..."
```

## Useful openers for Part 3

```
"I think most people would agree that..."
"In general, society tends to..."
"From a societal perspective, ..."
"It seems to me that the main reason is..."
"I believe governments have a responsibility to..."
```

## What to avoid

- `I personally...` — this is Part 1 language
- Short answers (2–3 sentences) — give as much as possible
- Memorised or scripted responses — examiners detect these

## Extending your answer

If you run out:
- Add a comparison: `This is different from the situation in [context]...`
- Add a consequence: `The long-term impact of this would be...`
- Add a concession: `Of course, there are exceptions, for example...`

## Mini practice

**Prompt:** "Do you think it's more important for governments or individuals to solve environmental problems?"

Practice using the structure above:
1. State a position (society/government level)
2. Give one reason
3. Add one implication or example
4. Optional: concede one point
