# Error Corrections — Active Review Inbox

Each note stores one learner sentence or short response:

- Front: the exact original wording;
- Back: the corrected version and concise error analysis.

These notes stay here until the user reviews them. The direct command
`整理 anki` counts as approval, creates the Anki cards, and moves confirmed source
notes to `archive/`.

This workflow does not create Vocabulary, Expression, Sentence Pattern, or Idea
notes automatically.
