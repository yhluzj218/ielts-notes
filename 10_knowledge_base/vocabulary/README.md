# Vocabulary Active Inbox

This folder contains user-selected single-word notes that have not yet been
exported to Anki.

Create notes with:

`08_ai/anki/vocabulary_intake_workflow.md`

The intake workflow researches and stores the word here but does not create an
Anki card. The `整理 anki` workflow later creates the card and moves its canonical
source note to `archive/`.

Search both this folder and `archive/` before creating a word.
