(still zero new data this sprint before today). Reading is "maintain only" per the
profile, but it's had no timed session logged all sprint either — one passage today
is upkeep, not a new priority.

**Exercise · D2 — Listening (90 min)**

Same structure as Sprint-001 §5 Tue 7/7 protocol:
1. Before audio (10 min): read all questions, underline differentiating keywords,
   predict word type for Note Completion items
2. Listen and answer (35 min): no writing after the first keyword; watch for
   qualifier traps, self-corrections, conditional branches
3. Transcript analysis (45 min): for every wrong answer, find the correct answer,
   find the eliminating words, classify error type (see `connie_profile.md` §7.5
   categories)

**Exercise · Reading (60 min)**

One Cambridge passage/section, timed (~20 min per passage + review). Apply the
Matching Features SOP from `connie_profile.md` §6.2: split each statement into
2–4 conditions, verify all of them, don't stop at the first keyword match.

**Review**

- [ ] Reading: for each wrong answer, was a condition missed, or was it inference
