# WR007 — Task 1 Letter: Structure and Register Rules

## Core Rule

Every bullet point in the task gets its own **separate paragraph**. Do not merge bullets.

Task 1 GT letters have three bullet points → three body paragraphs minimum.

## Salutation and Closing Rules

| Opening | Closing | When to use |
|---|---|---|
| `Dear Sir/Madam,` | `Yours faithfully, [Full name]` | Formal — you don't know the person |
| `Dear Mr/Ms [surname],` | `Yours sincerely, [Full name]` | Semi-formal — you know the name |
| `Dear [first name],` | `Love, [name]` / `Take care` / `All the best` / `Hope to catch up soon` | Informal — friend/family |

**Memory aid:** `faithfully` with `Sir/Madam` (no name known). `Sincerely` with a name.

## Tone guidelines

| Letter type | Tone |
|---|---|
| Complaint | Can be firm/direct — no need to over-apologise |
| Request | Polite but direct |
| Invitation / thanks | Warm |
| Friend letter | Natural, personal, conversational |

## Informal letter tips (Matt, 2026-06-26)

- When writing to a specific friend: imagine that real person. Add personal details.
- Make the tone sound like YOU talking to THEM — not like a template.
- Do NOT end with `Regards` in an informal letter — too distant for a friend.
- `Keeping in touch` / `Hope to catch up soon` sound natural for friends.

## Word count

Do not stop exactly at 150 words. Write slightly more to give yourself room to score higher.

## Critical rule: stick to the bullets

Any information NOT required by the bullet points = negative marks.
- Wastes word count
- Increases chance of grammatical errors
- Directly penalises Task Achievement

## Structural template

```
[Opening: Dear [X],]

[Paragraph 1 — Bullet Point 1]

[Paragraph 2 — Bullet Point 2]

[Paragraph 3 — Bullet Point 3]

[Closing phrase,]
[Full name]
```
