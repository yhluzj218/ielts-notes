# P004 — `request` (No `for`) · `as a result of` (Not `in a result of`)

## Two Errors, One Card

Both involve preposition misuse in formal writing contexts.

---

## Rule 1: `request` Is Transitive — No `for`

`request` as a **verb** is transitive: it takes a direct object with **no preposition**.

❌ `requesting for a refund`
✅ `requesting a refund`

`request` as a **noun** pairs with `for`:
✅ `a request for a refund` (noun phrase)

**Pattern summary:**

| Form | Pattern | Example |
|---|---|---|
| Verb | `request + [noun]` | `I would like to request a refund.` |
| Noun | `a request for + [noun]` | `My request for a refund was ignored.` |

---

## Rule 2: `as a result of` (Not `in a result of`)

`in a result of` does not exist in English. The correct fixed phrase is `as a result of`.

- `as a result` = therefore / consequently (standalone)
- `as a result of + [noun phrase]` = because of [noun phrase]

❌ `In a result of poor planning, the project failed.`
✅ `As a result of poor planning, the project failed.`
✅ `The project failed. As a result, they lost funding.`

---

## Good examples

✅ `I would like to request a meeting at your earliest convenience.`
✅ `She submitted a request for more information.`
✅ `As a result of the new policy, emissions have decreased.`
✅ `The road was closed. As a result, traffic was diverted.`

## Reusable patterns

```
request [noun]: request a refund / request a meeting / request assistance
as a result of [noun phrase]: as a result of increased demand / as a result of the policy
[Cause]. As a result, [effect].
```

## Mini practice

Fix the errors:

1. `She requested for permission to leave early.`
2. `In a result of the storm, the event was cancelled.`
3. `He made a request for additional resources.` ← Is this correct?

**Answers:**
1. She **requested** permission to leave early. (remove `for`)
2. **As a result of** the storm, the event was cancelled.
3. ✅ Correct — `a request for` + noun is fine.
