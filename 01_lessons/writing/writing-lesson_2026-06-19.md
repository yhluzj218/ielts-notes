## Task 1 重點回饋（火車退票信）

| 主題 | 老師的 Feedback |
|------|----------------|
| 時態 | `last week` 要用過去式 → `I purchased`，不是 `I have purchased` |
| 分段 | 每個 bullet point 各開一段，不能全部擠在一起 |
| 字數 | 不要剛好停在 150 字，多寫一點才有機會拿高分 |
| 詞性 | `loyalty customer` → `loyal customer` |
| staff 不可數 | `staff` 不可數；要數人用 `staff member` |
| 請求句文法 | `requesting for a refund` → `requesting a refund`（不需要 for）|
| 單複數 | `a refunds` → `our refund` |
| 正式結尾 | Dear Sir/Madam → `Yours faithfully` + 全名（名字 + 姓）；Regards 適合 email，正式信件不夠 |
| 投訴語氣 | 投訴信語氣可以稍強硬，不需要太客氣。老師建議句型：`I would suggest training your staff to deal with issues in a more efficient manner and to listen carefully to customers' concerns.` |
| 詞彙肯定 | `providing any further assistance`、`my plan was delayed as a result of it` — 老師說「really nice vocabulary, good enough for 7」|

## Task 2 重點回饋（報紙/書籍議論文）

| 主題 | 老師的 Feedback |
|------|----------------|
| 最大問題：題目理解 | `agree/disagree` ≠ `positive/negative attitude`——這題問你同不同意，不是叫你說好壞 |
| 改寫題目不能改意思 | 題目說 `nobody`（強詞），改成 `less people` → 意思變了，立場就跑掉了 |
| 題目時態 | 題目說的是未來（will stop），不能用現在式回答，方向會偏掉 |
| 觀點要在開頭說 | 不要把主要意見留到最後——要一開始就表態，再給理由 |
| Task 2 標準結構 | Intro（改寫題目 + 表態）→ Body 1（理由一）→ Body 2（理由二）→ Conclusion（總結）|
| although 用法 | `although` 放句首製造對比：`Although I like pizza, I try not to eat it`——不能放在句中亂接 |
| deliberate 用錯 | 想說 discuss/explain 卻用了 `deliberate`——硬用高級詞反而扣分 |
| information 不可數 | `information` 永遠不加 S |
| social medium | `in social medium` → `on social media` |
| 觀點句寫法 | `my point of view agrees` → `I certainly agree` 或 `my point of view is that...` |
