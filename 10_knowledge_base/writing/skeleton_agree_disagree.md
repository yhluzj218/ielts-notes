# Skeleton — Agree / Disagree

> Last updated: 2026-07-01

---

## 辨識規則

**只看最後一句指令：**

| 指令關鍵字 | 判斷 |
|---|---|
| `Do you agree or disagree?` | ✅ Agree/Disagree |
| `To what extent do you agree?` | ✅ Agree/Disagree |
| `Is this positive or negative?` | ✅ Agree/Disagree |
| `Do advantages outweigh disadvantages?` | ✅ Agree/Disagree |
| `Discuss both views` | ❌ 不是這個題型 |

**常見陷阱：** 題目有兩個對立面 ≠ 一定是 Discuss Both Views。指令只問你的立場 → 就是 Agree/Disagree，兩個 body 都支持同一個立場。

**立場規則：** Intro 就表態。不騎牆。

---

## Variation A — Two Reasons（兩個獨立理由）

最標準的結構。兩個 body 各給一個理由，都支持同一立場。

```
Intro:   Paraphrase + clear position (agree / disagree)
Body 1:  Reason 1 + development chain
Body 2:  Reason 2 + development chain
Concl:   Restate position
         （可加讓步：While it is true that [對方說法], [你的立場] far outweighs...）
```

**真實例子 →** [Language Loss](../../02_writing/task2/models/language-loss_model_2026-06-26.md)

題目主張：語言消失不要緊，全球溝通更方便。
立場：強烈不同意。

```
Intro:   語言消失 = 文化損失 → I strongly disagree, as it overlooks two fundamental values
Body 1:  Reason 1 — 語言是知識與文化的載體（例：古埃及文）
Body 2:  Reason 2 — 語言捕捉其他語言無法複製的概念（例：愛斯基摩人的雪）
Concl:   While it is true that 共同語言方便溝通，
         this convenience is far outweighed by what would be permanently lost
```

## Variation B — Refute Two Claims（逐一反駁題目的兩個主張）

題目本身包含兩個具體主張。你不同意兩個 → 兩個 body 各攻一個主張。

```
Intro:   Paraphrase + firmly disagree for two reasons
Body 1:  Refute Claim 1 + development chain
         （可加 body 內讓步：Even though some X, there will still be Y）
Body 2:  Refute Claim 2 + development chain
         （可加 body 內讓步：It is true that X, despite this Y）
Concl:   Restate disagreement（點名反駁了哪兩個主張）
```

**何時用：** 題目說「因為 X，所以 Y 會發生」→ 分別反駁 X 和 Y。

**真實例子 →** [Print vs Digital](../../02_writing/task2/models/print-vs-digital_model_2026-06-19.md)

題目兩個主張：(1) 沒有人會買實體報紙/書 (2) 所有內容都能免費看
立場：不同意兩件事。

```
Intro:   改寫題目 → However, I firmly oppose this idea for two important reasons
Body 1:  Refute 主張1 — 仍有很多人偏好實體（例：學生需要手寫批注）
         Even though some will switch to digital, there will still be...
Body 2:  Refute 主張2 — 並非所有內容都會免費（例：NYT / Economist paywall）
         It is true that some may be free, despite this...
Concl:   I strongly disagree with both [主張1] and [主張2]
```

## Variation C — Negative Case + Positive Case（推翻舊 + 建立新）

題目主張「應該從 A 轉向 B」。你完全同意 → Body 1 論述為何 A 沒必要，Body 2 論述 B 的好處。

```
Intro:   Paraphrase + completely agree
Body 1:  [負面論證] 為什麼舊做法（A）沒必要 / 無效
Body 2:  [正面論證] 新做法（B）能帶來什麼
Concl:   Restate position（可加對比：A → B）
```

**何時用：** 題目問「應該從 X 轉向 Y」，你完全同意。

**真實例子 →** [Skills vs Facts](../../02_writing/task2/models/skills-vs-facts_model_2026-07-01.md)

題目主張：學校應從教 facts 轉向教 skills。
立場：完全同意。

```
Intro:   As information is accessible online, schools should move away from facts → I completely agree
Body 1:  [負面論證] 為什麼不需要在學校教 facts
         - Reason 1: 多媒體課程已在網路上（例：YouTube）
         - Reason 2: 學生有能力自己查資料 → 可當作回家作業
Body 2:  [正面論證] 學校應該改教什麼
         - Practical skills: research, presentations, programming → 職場需求
         - Social skills: teamwork, negotiation, delegation → emotional intelligence
Concl:   Classroom = skills; facts = study at home via Internet
```

## Skeleton 自我檢查

- [ ] 題型判斷正確？（依指令，不是依內容）
- [ ] Intro 有清楚的立場句？
- [ ] Body 1 和 Body 2 各有一個主要想法，不重疊？
- [ ] Conclusion 只重申立場，沒有加新論點？
