# L005 — Prediction Before Audio: Predict the Word Type

## Core Rule

Before the recording starts, look at each gap and ask: **"What type of word goes here?"**

- Is it a number? A date? A noun? An adjective?
- Is it singular or plural?

This gives your listening a **target** — instead of listening to everything, you listen for one specific type of word.

## Why it matters in IELTS

Prediction transforms passive listening (trying to follow everything) into active listening (waiting for one specific type of answer). This is especially effective for Note Completion in S4.

## How to use the pre-read time

For each gap, note:

| Question type | What to predict |
|---|---|
| `The museum opens at _____` | Time → number |
| `Visitors must bring their _____` | Thing → noun; check if singular or plural |
| `The temperature was _____ than expected` | Degree → adjective (comparative) |
| `The fee is $_____` | Number |
| `She is a _____ professor` | Job/type → adjective or noun |

## Strategy

1. During pre-read: write a tiny label next to each gap (`#`, `n`, `adj`, `pl`)
2. While listening: focus your attention on the predicted word type
3. If the word type matches → write; if it doesn't → wait (might be a distractor)

## Mini practice

Predict the word type for each:

1. `The bridge was built in _____.` → ?
2. `The programme is suitable for _____ students.` → ?
3. `The venue can hold up to _____ people.` → ?

**Answers:**
1. Year → number
2. Type of student → adjective (or noun modifier)
3. Capacity → number
