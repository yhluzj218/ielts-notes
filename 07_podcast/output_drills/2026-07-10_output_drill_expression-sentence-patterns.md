Parenting, education, responsibility, and learning through experience

Core Patterns:
1. It is true that [X]. However, I completely disagree with the idea that [Y].
2. As well as being [adjective], I would argue that [Y].

Target Vocabulary / Expressions:
1. shoulder a huge responsibility
2. by no means an easy task
3. trial and error
4. set someone up for success
5. set the bar higher

The fourth expression is:

set someone up for success.

[pause 3 seconds]

If something sets someone up for success, it gives them the conditions they need

The fifth expression is:

set the bar higher.

[pause 3 seconds]

To set the bar higher means to raise the standard.

You can use it when current expectations are too low.
