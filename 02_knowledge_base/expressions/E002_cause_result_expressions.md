# E002 — Cause and Result Expressions

## Core Rule

Different expressions require different grammatical structures. Mixing up noun phrases and clauses is the most common error.

## Cause Expressions

| Expression | Takes | Example |
|---|---|---|
| `because` | clause (subject + verb) | `because demand has increased` |
| `due to` | noun phrase only | `due to increased demand` |
| `as a result of` | noun phrase only | `as a result of the new policy` |
| `owing to` | noun phrase only | `owing to poor planning` |
| `since` | clause | `since the law was changed` |
| `due to the fact that` | clause | `due to the fact that demand has increased` |

**Critical:** `due to` = preposition → noun phrase only (never a clause with subject+verb)
See [G011](../grammar/G011_due_to_clause.md)

## Result Expressions

| Expression | Pattern | Example |
|---|---|---|
| `As a result,` | Sentence opener | `As a result, pollution levels have dropped.` |
| `as a result of` | `as a result of [noun], [result]` | `As a result of the policy, emissions decreased.` |
| `Therefore,` | Sentence opener | `Therefore, further action is required.` |
| `Consequently,` | Sentence opener | `Consequently, many people lost their jobs.` |
| `This has led to` | `This has led to [noun]` | `This has led to a significant increase in unemployment.` |
| `This means that` | `This means that [clause]` | `This means that fewer people can afford housing.` |

**`in a result of` does not exist.** → Use `as a result of`

## Useful sentence patterns

```
[Cause clause], which has led to [result noun].
Rising costs have led to a decline in consumer spending.

[Cause]. As a result, [effect].
Demand increased rapidly. As a result, prices rose.

Due to [noun phrase], [main clause].
Due to rising demand, prices have increased significantly.

Because [clause], [main clause].
Because demand has increased, prices are now higher.
```
