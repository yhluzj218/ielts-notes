# Sentence Patterns by Skeleton Position

> 按 skeleton 位置分類。每個句型附來源範文。
> 不是要背整句——是要知道「這個位置可以用什麼結構」。
> Last updated: 2026-07-01

---

## Introduction

### 第一句 — 改寫題目 / 帶出背景

```
As [background fact], it can be argued that [restate the debate].
```
> As information is now so accessible online, it can be argued that schools should move away from fact-based teaching.
> 來源：[Skills vs Facts](../task2/models/skills-vs-facts_model_2026-07-01.md)

```
Some argue that [opposing view / topic claim].
```
> Some argue that language loss is not a matter of concern, as fewer languages would make communication more convenient.
> 來源：[Language Loss](../task2/models/language-loss_model_2026-06-26.md)

```
Some people have suggested that [restate the claim].
```
> Some people have suggested that people will stop purchasing physical newspapers since everything will be available online for free.
> 來源：[Print vs Digital](../task2/models/print-vs-digital_model_2026-06-19.md)

---

### 第二句 — 表態

```
I completely agree with this idea.
I strongly disagree with this view, as it overlooks [reason].
However, I firmly oppose this idea for two important reasons.
This essay will argue that [your position + one-line reason].
This essay will examine both perspectives before presenting my own view.
```

**Agree/Disagree：** 第二句就說清楚你同意或不同意。不要拖到結尾才表態。

---

### Problem + Solution 專用

```
[Paraphrase problem]. This essay will explore the main reasons behind this trend and suggest several measures to address it.
```

❌ 不能寫：`I will show some reasons and solutions in the following paragraph.`（太模糊）

---

## Body — 段落開頭 [point]

### 預告多個理由

```
There are two main reasons why I believe that [your position].
```
> 來源：Skills vs Facts

### 開頭直接給主旨

```
First and foremost, [main idea].
Firstly, [main idea].
A shift away from [X] would allow [Y] to [do something].
Another important issue with [the proposed idea] is that [new point].
```

### 展開兩個子主題（同一段內）

```
With regard to [subtopic], a modern approach to [X] would put more emphasis on [Y].
```
> 來源：Skills vs Facts — 用來在 Body 2 內從 practical skills 轉到 social skills

### 定義 / 擴大概念

```
Firstly, [X] is far more than [narrow definition] — it is [broader definition].
```
> Firstly, language is far more than a tool for daily communication — it is the medium through which human knowledge and cultural heritage are preserved.
> 來源：Language Loss

---

## Body — [explain] 說明原因

```
This is because [reason].
It is clear to see that [supporting fact or trend].
Useful [adjective] skills might include [list].
```

---

## Body — [support] 舉例

```
[Specific example], for example, [detail].
[Noun] is a powerful example: without [condition], [result] would have remained [adj].
For instance, [group] have [specific detail] — [what this shows].
```

> YouTube, for example, contains videos that cover the full range of school subjects.
> 來源：Skills vs Facts

> The ancient Egyptian hieroglyphic script is a powerful example: without scholars painstakingly decoding this extinct writing system, the knowledge would have remained permanently inaccessible.
> 來源：Language Loss

> The Inuit people, for instance, have dozens of distinct words for different types of snow.
> 來源：Language Loss

---

## Body — [result] 後果 / 意義

```
In my opinion, this [X] could be done as [Y], which would [consequence].
Competence in these areas is in high demand in today's job market.
This way of [doing X] could develop [quality] and help all [people], regardless of [factor], to [achieve goal].
No single [X], however [adj], could fully [V].
```

> In my opinion, this research could be done as homework, which would remove the need for the teaching of facts in the classroom.
> 來源：Skills vs Facts

> This way of teaching could develop students' emotional intelligence and leadership qualities, and help all children, regardless of academic ability, to reach their potential.
> 來源：Skills vs Facts

---

## Body — 讓步句（段落內）

在 body paragraph 內承認對方有一點道理，然後轉回自己的論點。

```
Even though [concession], there will still be [your point].
It is true that [concession], despite this it seems clear that [your point].
It seems unlikely that [the claim], because [your reason].
```

> Even though certain people will switch to digital-only in the coming decades, there will still be a significant part of society who will use hard copies.
> 來源：Print vs Digital

> It is true that certain organisations may choose to make content free, despite this it seems clear that people will need to pay to access valuable information.
> 來源：Print vs Digital

---

## Conclusion

### 重申立場

```
In conclusion, I believe that [restate position], while [contrast].
In conclusion, [restate your main position in different words].
To sum up, despite [concession], I strongly disagree with both [claim 1] and [claim 2].
```

> In conclusion, I believe that classroom teaching should prioritise skills and interpersonal relationships, while factual knowledge can be studied at home via the Internet.
> 來源：Skills vs Facts

> To sum up, despite the increasing trend towards digital media, I strongly disagree with both the idea that physical newspapers will no longer be bought and the reason given that all information will be available for free.
> 來源：Print vs Digital

### 讓步 + 反駁（用於結論段）

```
While it is true that [opposing point], this [convenience / benefit] is far outweighed by [your reason].
```
> While it is true that sharing a common language would ease certain practical interactions, this convenience is far outweighed by what would be permanently lost.
> 來源：Language Loss

### 行動句（可選）

```
Rather than [accepting X / doing X], we should [positive action].
```
> Rather than accepting this erosion as inevitable, we should actively work to document and revitalise endangered languages.
> 來源：Language Loss

---

## ⚠️ 錯誤陷阱

| 陷阱 | 正確寫法 |
|---|---|
| `Although X, however Y` (G010) | `Although X, Y` 或 `X. However, Y.` |
| 結論加新論點 | 只重申立場，不加新內容 |
| `I will show some reasons...` | 點名具體內容：`...explore the main reasons... and suggest measures...` |
| Body 1 說「some people think X, however this is wrong」 | Discuss Both Views 的 Body 1 必須認真論述對立觀點 |
