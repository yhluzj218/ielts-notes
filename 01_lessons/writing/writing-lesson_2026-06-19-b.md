## Task 1 重點回饋（訓練課程回饋信 v2）

| 主題 | 老師的 Feedback |
|------|----------------|
| feedback 不可數 | `a short feedback` → `some feedback` 或 `a piece of feedback` |
| 時態 | `I have participated in` → `I participated in`（last weekend = 過去式）|
| 主謂一致 | `There is several` → `There are several` |
| 形容詞 vs 副詞 | `made it easily` → `made it easy`（easy 是形容詞，不是副詞）|
| 過去被動 | `we been separated` → `we were separated into different groups` |

| 語氣定位 | Dear Will = semi-formal，OK；正式是 Dear Sir/Madam；非正式是和朋友說話的語氣 |
| 內容要更寫實 | 加入跟 McDonald's 工作實際相關的細節，例如：`As we frequently deal with difficult customers in the evening, the session on managing challenging situations was particularly useful.` |
| Bullet 3 要展開 | 說了「對經理不相關」，但沒有解釋**為什麼**不相關；要補一句理由 |

## Task 2 重點回饋（學費議論文 v2）

| 主題 | 老師的 Feedback |
|------|----------------|
| due to 用法 | `due to` ≠ `because`；若要用，要說 `due to the fact that`；直接用 `because` 更安全 |
| reasons 數量要一致 | 說 `two main reasons` 但寫了三個 body paragraphs → 要一致：說幾個理由就寫幾段 |
| end up 用法 | `end up going for work` → `end up working rather than continuing their studies`（work 是動詞，不能當名詞這樣用）|
| idea 不要只點到就跳走 | Para 2 說了「是負擔」，馬上跳到創業者例子，中間缺少解釋（為什麼是負擔？具體說明：例如學費是三個月薪資、學生沒有收入等）|
| example 後要收尾 | 不要用 example 結束段落——要說明這個例子跟題目的關係，例如 Para 3 的 Cambridge 例子後要補：「education is the only way to reverse the class divide」|
| Para 4 開頭要清楚 | 想法太複雜，讀者看不懂；老師建議開頭先用簡單句：`Education is so important that it should not be treated like other products.` 先讓讀者理解方向，再解釋細節 |
| a part of a portion | 同義重複——只用一個：`part of it` 或 `a portion of it` |
| 正面回應題目前提 | 題目說「degree 對個人好處大於社會」→ 要直接回應：可以說 `society benefits as much as students`（醫生、老師例子）或 `society benefits more`；用這作為一個論點比三個散落的論點更有力 |
| 想法不要太複雜 | 考場只有 5 分鐘想，花太多時間想出的複雜想法反而更難清楚表達；簡單想法講清楚 > 複雜想法講不清楚 |
