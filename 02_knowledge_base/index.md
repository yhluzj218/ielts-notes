## How to Use This Knowledge Base

### For NotebookLM
Upload all `.md` files in `02_knowledge_base/` as a source. Ask NotebookLM to:
- Generate a quiz from grammar cards
- Explain a specific rule in the format of a podcast
- Create flashcards from the expression cards

### For AI Lesson Generation
Provide this `index.md` as context, then ask the AI to:
- Design a lesson focused on 🔴 Critical cards
- Create a practice exercise using [card ID]
- Link errors from a writing draft to their knowledge cards

### For Self-Study
Start with all 🔴 Critical cards. Then move through High → Medium by category.
