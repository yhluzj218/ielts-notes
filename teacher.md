# 老師的核心原則（我兩位老師教的重點）

## 跨老師、最重要的共識

- 我不缺詞彙。兩位老師、口說和寫作都確認過這點。
  - 口說老師：「我的直覺是你不缺詞彙，是你以為自己缺。」
  - 寫作老師：「詞彙是你寫作最好的部分，我從沒提過它。」
- 硬塞高級詞彙反而扣分（例：deliberate 用錯、口說為想一個字而卡住）。
- 我已經考過 7.5，能力在那裡。現在是「找回手感」，不是「補能力」。

## 口說原則

- 別過度思考（老師反覆強調 don't overthink）。
- Part 1：直接回答、答完就停、只答被問的問題；長短看題目決定。
- Part 2：
  - 一定要講滿 2 分鐘（提早停比講太多更糟）。
  - 用題目本身搭結構（開頭=這是什麼 / 中段=細節 / 結尾感受）。
  - 用 linking phrase 串接，但「不需要連成完整故事、不需要結論」。
  - 準備時間（1 分鐘）也要用來「篩選好不好講的題材」，
    避開詞彙門檻高的主題。
- Part 3：講「一般人 / 社會普遍現象」，不要講自己（這是 Part 1/2 才做的）。
- 高分關鍵 = 傳達大量自然的資訊與細節，不是用複雜語言。
  簡單語言反而溝通更好。

### 不要做的事（2026-06-01 起，Matt 明確強調）
- 不要跟讀/仿效母語者：你的英文已夠好，複製反而干擾自然溝通
  （「It's got to be your communication during the exam, not imitating a native speaker.」）
- 不要看社群媒體/YouTube 的 IELTS 備考內容（考官不想看到那些「技巧」）
- 不要背稿/背答案：考官故意問意外問題，看你怎麼應對；答案太像準備好的會扣分

### 連讀 / 自然語調（試聽課點出）
- 「You're pronouncing words too individually / too correctly.」
- 例：`I would like to go` → `I'd like to go`（母語者不把每個字都發清楚）
- 這比個別單字發音更影響分數，也影響聽力理解

### Shadowing（跟讀）
- 有幫助，但要先上完「句子語調課（intonation lesson）」再做
- 先了解方法，跟讀才有效；學會前不要盲目跟讀

### 亞洲 vs 歐洲考生平衡
- 亞洲：大量備題/背答案 → 聽起來太機器人、不自然
- 歐洲：只靠放鬆說話 → 有時沒東西可說
- 最佳平衡：準備常見主題的想法，但不背稿；自然說話

### 備考時間策略
- 現在：重點改文法 + 發音（流暢度暫時下降是正常的，不要擔心）
- 考前：文法/發音穩定後，只需要思考題目和想法

## 寫作原則

- 低分主因是「沒答到題 + 結構不對」，不是英文不好。這些是技術問題，
  一講就能改、能短期跳分。
- 先修「切題」和「結構」，不要先談詞彙文法。
- Task 2 讀題：
  - 看清楚問法：agree/disagree ≠ positive/negative
    （agree/disagree 最常考，約佔 40–50%）。
  - 改寫題目時不能改掉原意（題目用 nobody/all 這種強度字，
    不能改寫成 less/some，否則立場就變了）。
  - 要明確表態 + 給理由，不要只列優缺點。
  - 觀點要在開頭就說，不要埋到最後。

### Task 1 信件原則（2026-06-19）
- 每個 bullet point 各開一段，不能全部擠在一起。
- 正式結尾規則：Dear Sir/Madam → Yours faithfully + 全名；Dear + 名字 → Yours sincerely。
- 投訴信語氣可稍強硬，不需要太客氣。
- 字數不要剛好停在 150 字，多寫一點才有機會拿高分。

### Task 2 原則（2026-06-19）
- although 用法：放句首造對比（Although I like pizza, I try not to eat it）；句內不再加 however 或 but，兩者只能選一個。
- 段落展開結構：點出觀點 → 解釋為什麼 → 給例子 → 說明例子跟題目的關係（不能用 example 結束段落）。
- 題目有 "because + 前提" 子句 → 要直接回應那個前提，不只是說「後果不好」。
- `due to` ≠ `because`；要接子句時用 `because` 或 `due to the fact that`。
- 說了幾個 reasons，body paragraph 就寫幾段，數量要一致。
- 考場只有 5 分鐘想題：簡單想法說清楚 > 複雜想法說不清楚。

### 練習建議（2026-06-19）
- 練寫作至少要保留 1 小時，15 分鐘不夠，省下來做口說和聽力。
