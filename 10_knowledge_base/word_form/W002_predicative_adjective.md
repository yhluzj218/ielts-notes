# W002 — Predicative Adjective After `make it / find it / consider it`

## Core Rule

After verbs like `make`, `find`, `consider`, `keep`, `leave` followed by `it`, use an **adjective** — not an adverb.

Structure: **`[verb] + it + [adjective]`**

> `make it easy` (not `make it easily`)
> `find it difficult` (not `find it difficultly`)

**Why:** `it` is the object; the adjective describes `it`. Adverbs modify verbs, not nouns.

## Why it matters in IELTS

These constructions are common in Task 2 when discussing effects: "Technology makes it easier to...", "People find it difficult to...". Using an adverb here is grammatically wrong and clearly visible.

## Common mistakes

❌ `Technology made it easily for people to communicate.`
❌ `She found it difficultly to adapt.`
❌ `They made it cost effectively by using cheaper materials.`

## Good examples

✅ `Technology made it easy for people to communicate.`
✅ `She found it difficult to adapt to the new environment.`
✅ `They made it cost-effective by using cheaper materials.`
✅ `The app makes it possible to track your spending.`

## Reusable patterns

```
make it + [adj]: make it easy / possible / difficult / clear / affordable
find it + [adj]: find it hard / challenging / important / useful
consider it + [adj]: consider it necessary / reasonable / fair
keep it + [adj]: keep it simple / brief / relevant
```

**Followed by `to + infinitive`:**
✅ `It makes it easier to find information.`
✅ `She found it impossible to concentrate.`

## Mini practice

Fix the adjective/adverb error:

1. `The new system makes it easily to submit applications.`
2. `Many students find it difficultly to manage their time.`
3. `Online shopping has made it conveniently to buy goods.`

**Answers:**
1. makes it **easy** to submit applications
2. find it **difficult** to manage their time
3. made it **convenient** to buy goods
