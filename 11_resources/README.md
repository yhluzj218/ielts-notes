# 11_resources — IELTS 教材與學習工具收集

存放 Connie 收集的 IELTS 教材、學習工具、網站等資源紀錄。

## 用法

對 Claude 貼 URL 或截圖，說「記錄這個」（或類似指令），Claude 會：

1. URL → 讀取網頁，判斷是教材還是工具，整理摘要後寫入 `resources.md`
2. 圖片 → 存到 `images/`，描述內容後寫入 `resources.md`

## 紀錄格式（resources.md）

每筆資源一個 `###` 區塊，新的加在最上面：

```markdown
### 資源名稱
- 類型：教材 / 工具 / 網站 / 課程 / 頻道
- 連結：URL（或 images/ 檔案路徑）
- 用途：對哪個科目或技能有幫助（Writing / Speaking / Listening / Reading / 綜合）
- 摘要：一兩句說明這是什麼、為什麼值得記
- 記錄日期：YYYY-MM-DD
```
