# Stage 2 Output Drills

Generated ElevenLabs-ready Stage 2 scripts are stored here:

`YYYY-MM-DD_output_drill_<topic-slug>.md`

Each file records the context source and contains an `ElevenLabs-ready Script`
section. Stage 2 never selects the Core Pattern, Model Sentence, or Vocabulary /
Expressions; those remain required user inputs.

The spoken script begins without `[TEACHER]` or other speaker labels. It opens with
a warm welcome, closes by thanking the listener and introducing the next song or
today's chant, and uses pause durations only in multiples of three seconds.
