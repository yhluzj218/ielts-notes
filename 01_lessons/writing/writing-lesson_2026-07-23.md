## 重點 1 — 複習寫作固定流程：先判斷 Type

- 拿到題目先做的事：**判斷 type**（不是先想內容）。
- Opinion 型的流程：Type → 你的 Opinion → 兩個支持 Opinion 的 Reasons。
- Introduction 三件事（Opinion 型）：Introduce topic → Opinion → Reasons for opinion。

## 重點 2 — Problem and Solution 型 vs Two-part 型的差異（Joe 直接教學確認）

- **Two-part**：兩個問題之間**沒有關聯**（"there is no connection between the two questions"）。
- **Problem and Solution**：兩者有因果關聯——先講一個 problem，再講對應的 solution。
- 這個判斷邏輯與知識庫既有內容一致（`10_knowledge_base/writing/skeleton_two_part.md`：兩個問題「不一定是原因+解法，可以是任意兩個不同角度的問題」）——**這是 Joe 本人對既有規則的第一次直接口頭確認**，不是新規則，但值得記一筆來源。

## 重點 3 — Problem and Solution 的 Introduction 三件事

- Introduce the topic（不能只是把題目改幾個字重寫一次——"try not to just copy the question and change some of the words. Try and really think about how would you talk about this topic."）
- Problem（點名一個問題）
- Solution（點名一個解法）

## 重點 4 — 新的 Body 段結構示範：單一 Problem / 單一 Solution，各自獨立展開（新 variation）

這堂課示範的結構跟知識庫現有的 Problem+Solution 兩個 variation（Paired：每段一組因果；Separated：Body1 全部原因、Body2 全部解法）都不同——這次是：

```
Body 1 = 一個 Problem（陳述 → Explain 為什麼是問題 → Example）
Body 2 = 那個 Problem 對應的 Solution（陳述 → Explain 怎麼解決 → Example）
```

只處理**一組** problem-solution，不是兩組。已補到 `10_knowledge_base/writing/skeleton_problem_solution.md` 作為 Variation C。

- Joe 重申核心規則：「There must be one main idea per paragraph.」
- Joe 對選題的建議：**沒有「錯的」problem，選什麼都可以，重點是簡單好寫**（"there's no wrong problem... my advice is just keep it simple. It's much easier to write if you just keep the problem simple."）——呼應 Joe 一貫的「不要為了複雜而複雜」原則，這次套用在「選題」而不是「選詞彙/句構」上。
- 課堂練習時我第一次選的 problem 太籠統（"surrounding area, dirty"——沒說動物做了什麼造成的），Joe 糾正：要說清楚**動物做了什麼**造成這個問題。修正後定案：Problem = 動物產生的排泄物（waste produced by the animals）→ 造成社區異味（smells bad for the neighborhood）；Solution = 立法加強廢棄物管理（pass a law for better waste management）。

## 重點 5 — Introduction 練習句（課堂共同產出）

> "With many saying it is cruel to put animals in cages and have tourists stare at them all day."

- 這句是 Introduce the topic 的部分，用讓步／爭議點切入（不是直接改寫題目）。Joe 對這句沒有標記文法問題，只在我猶豫某個字要不要用時說：「try to avoid not spelling things correctly, but equally, if you can't think of a different word, just use the word, you know. It's okay.」——呼應「不要為了怕拼錯字而迴避正確的字」。
