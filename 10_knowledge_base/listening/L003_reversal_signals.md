# L003 — Reversal Signals: The Answer Just Changed

## Core Rule

When you hear a **reversal signal**, the speaker is cancelling or contradicting what they just said. The answer is what comes AFTER the signal.

> **Reversal signals:** `but / however / actually / not anymore / used to / originally / instead / in fact / on second thought`

## Why it matters in IELTS

Reversal signals are used deliberately in IELTS to create answer-switching moments. Missing one means choosing the wrong answer even when you correctly heard both options.

## Signal words and what they do

| Signal | Meaning |
|---|---|
| `but / however` | contrasts — what follows is the real answer |
| `actually / in fact` | corrects — speaker changes their mind |
| `not anymore / no longer` | reverses — the old situation is now false |
| `used to` | signals the PAST state (not the current state) |
| `originally / at first` | signals what WAS true, not what IS true |
| `instead` | replaces — the previous option is cancelled |
| `on second thought` | reconsiders — speaker changes their mind explicitly |

## Strategy

1. Mark the answer options during pre-read
2. While listening: stay alert for reversal signals
3. When you hear one → the previous answer is likely wrong; listen for the new one
4. Common trap: writing the answer, then missing the reversal signal that cancels it

## Example

> "We'll hold the event on Saturday... actually, the venue has changed, so it'll be Sunday instead."

→ Wrong answer: Saturday
→ Correct answer: Sunday (the reversal signal `actually` + `instead` cancelled Saturday)
